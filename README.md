# Tic-Tac-Toe in the Browser

# Description

A Tic Tac Toe game made utilizing vanilla HTML, CSS, and JavaScript. A win condition may be reached in the game by getting three of your respective input (X/O) in a row or by filling in the four corner cells of the board. The game operates on a turn-by-turn basis, allowing X to place an input first, then O and so on until a win condition or tie is reached.

# How to Use

In-order to run the program, download the HTML, JS, and CSS files to your local machine from this repository. Then, double click the HTML file in your file explorer to open the HTML file and associated project within your local browser.
